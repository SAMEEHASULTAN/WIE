RelaunchHer

Empowering Women to Restart, Rebuild, and Rise

Overview

RelaunchHer is a platform designed to support women in restarting their careers after breaks. It connects users with job opportunities, mentorship, NGOs, and government schemes, helping them transition from career gaps to meaningful comebacks.

Problem Statement

Many women face career breaks due to personal, social, or economic reasons. Returning to the workforce becomes difficult due to:

Skill gaps
Lack of guidance
Limited access to opportunities
Confidence barriers
Solution

RelaunchHer provides a centralized platform where women can:

Discover job opportunities
Access mentorship guidance
Explore NGOs and government schemes
Get AI-powered career support
AI Integration

The platform leverages AI to provide personalized experiences:

Skill extraction from user input or resumes
Job recommendations using external APIs such as JSearch
Intelligent matching between user profiles and opportunities
Features
Job recommendation system
Mentorship support
NGO listings for women support
Government schemes awareness
Resume-based skill analysis
Simple and accessible user interface
Future Enhancements
AI-based resume builder for career gap users
Mock interview and communication training assistant
Financial literacy and micro-funding support
Location-based opportunity discovery
Verified and secure community networking platform
Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Python (Flask)
Database: SQLite (or relevant database)
APIs: JSearch API
